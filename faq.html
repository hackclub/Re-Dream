FAQ

Got questions? We've got answers!

Da Basics
What is Re-Dream?

Re-Dream is an opportunity to work on something you're truly passionate about and we will provide you with the budget to make your dream game just as good as you've imagined it!

Who can participate?

This event is open to anyone aged 13–18, regardless of skill level.

All this for free? Is this legit?

Yup, all for free! Hack Club is a 501(c)(3) non-profit organization, and we've run a bunch of events like this in the past including a 180-person hardware hackathon at GitHub HQ and a seven-day game jam in Shanghai, China.

Projects
I've never coded before. Can I still participate?

Yes, we want you to! This hackathon is for all skill levels. We'll be running workshops and support calls to help everyone learn together.

What can I make?

Any kind of technical project is allowed websites, games, hardware, anything!

How many projects can I make?

As many as you want! Projects only count toward your progress when they're in a usable state they don't need to be finished, but anything you submit should be demowable.

Can I submit an existing project?

Existing projects are allowed, but you can only submit time that was logged after February 22nd, 2026.

Does my project need to be on GitHub?

All projects must be open source on GitHub or another similar software forge.

Other
Can I use AI?

Use of generative AI for graphics and audio is prohibited. AI for other purposes is allowed, but we won't accept projects that qualify as "AI slop." You also shouldn't use AI to write updates or project descriptions.

"AI slop" means projects that feel like they have little human effort put in think "interactive periodic table" or "generic task management website." We reserve the right to decide whether a project qualifies. If your project is polished and has a real human touch, we're highly unlikely to reject it!

Can I submit to this event and another Hack Club program?

Sorry, nope! "Double-dipping" is not allowed. If you make a project for this event, you can only submit it here unless another program specifically states otherwise.

Can I get involved or contribute?

We love this question! We'd love more help running workshops, support calls, or even help organizing AMAs. If any of these interest you, let us know!

Still have questions?

Reach out at email@hackclub.com or #re-dream we'd love to hear from you!
